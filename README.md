# SOC Monitoring Dashboard with Splunk

A simple Security Operations Center (SOC) Monitoring Dashboard built using Splunk to monitor network traffic, detect malicious IPs, and visualize security events using SPL queries and CSV lookup tables.

---

## Overview
This project focuses on monitoring network security events and identifying malicious IP addresses through a centralized Splunk dashboard.

---

## Key Features
- Monitored allowed and blocked network traffic
- Detected malicious IP addresses using CSV lookup tables
- Visualized network activity and security events over time
- Displayed top malicious IP addresses
- Created interactive dashboards using SPL queries

---

## Technologies Used
- Splunk Enterprise
- SPL (Search Processing Language)
- CSV Lookup Tables

---

## Dashboard Components
-  Blocked vs Allowed Traffic Chart
-  Malicious IP Detection Table
-  Top Malicious IPs Visualization
-  Network Activity Timeline

---

## Purpose
This project was developed to practice:
- SOC Monitoring
- SIEM Concepts
- Log Analysis
- Threat Detection
- Security Visualization using Splunk
