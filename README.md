# soc-monitoring-with-splunk
A simple SOC Monitoring Dashboard built using Splunk to monitor network traffic, detect malicious IPs, and visualize security events using SPL queries and lookup tables

Overview
This project is a simple SOC Monitoring Dashboard developed using Splunk for monitoring network security events and detecting malicious IP addresses.

Features
Monitor allowed and blocked network traffic
Detect malicious IP addresses
Visualize network activity over time
Display top malicious IPs
Use SPL queries and CSV lookup tables
Technologies Used
Splunk Enterprise
SPL (Search Processing Language)
CSV Lookup Tables
Dashboard Components
Blocked and Allowed Traffic Chart
Detected Malicious IPs Table
Top Malicious IP Addresses Visualization
Network Activity Over Time
Purpose
This project was created for learning and practicing SOC monitoring and SIEM concepts using Splunk.
